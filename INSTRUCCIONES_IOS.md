# Guía de Notificaciones para Usuarios iOS (iPhone/iPad)

Apple tiene restricciones estrictas para las Web Apps (PWA). Para que las notificaciones funcionen, el usuario **debe** instalar la aplicación en su pantalla de inicio.

## Texto sugerido para compartir con usuarios:

---

### 🔔 ¿Cómo activar notificaciones en tu iPhone?

Para recibir alertas de nuevos pedidos, mensajes y avisos importantes, sigue estos 3 pasos:

**1. Abre en Safari**
Entra a la aplicación desde el navegador Safari (no uses Chrome ni otros en este paso).

**2. Agrega a Inicio (Obligatorio)**
Toca el botón **Compartir** (cuadrado con flecha ⬆️) y selecciona la opción **"Agregar al inicio"** ➕.

**3. Abre y Permite**
Cierra Safari. Busca el icono de la app en tu pantalla de inicio y ábrelo. Al iniciar sesión, te pedirá permiso para enviarte notificaciones. Toca **Permitir**.

---

## Requisitos Técnicos
- **iOS Version**: 16.4 o superior.
- **Navegador**: Safari (para la instalación).
- **Modo**: Debe correr como "Standalone" (desde el icono del home), no dentro de una pestaña de navegador.
